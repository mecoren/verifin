# 更新日志

本文件记录 Veri Fin 面向用户的版本变化，最新版本在前。仅收录用户可见的改动，纯内部重构 / 文档更新不在此列。

改动随做随记：做了用户可见的改动就往 `## [Unreleased]` 加一行；发版时把 `Unreleased` 改名为版本号加日期，再在其上新开一个空的 `Unreleased`。

## [Unreleased]

### 修复
- 修复**把一笔待报销支出的金额改到低于已报销/退款额**后，这笔支出会被错误计入「收入」、账户余额虚增的问题：净额现在钳制为不小于 0。
- 数据库因损坏或**降级安装了更旧版本**而打不开时，不再白屏卡死，改为显示明确的错误页——提示数据很可能仍在本机、**请勿清除应用数据**，并给出重装最新版的引导；绝不会自动删库。
- **恢复备份 / 重置数据 / 删除账本**现在在单个数据库事务内完成：万一中途进程被杀或写入出错，会整体回滚，不再可能留下「交易换了但账户没换」的半新半旧状态。
- WebDAV 请求增加连接与响应超时：当服务器无响应时不再无限期挂起，此前这会导致**自动备份此后被静默跳过、一直不再运行**（需重启应用才恢复）。

## [1.8.5] - 2026-07-07

### 优化
- 收支统计页图表的 x 轴日期改为**每 5 天显示一个**（1、5、10……），不再把整月日期挤成一团；手指滑动时数据气泡仍显示具体某天。首页概览卡月末多日时同样稀疏显示。
- 收支统计页下方**每日记录可点击**，点进即某天的交易明细（与首页日历点日期一致）。
- 概览卡自定义页布局调整：**卡片标题移到顶部**、各分区用卡片包起来、点槽位的**指标选择列表按类别分组**（本月 / 今日 / 本周 / 本年 / 总额 / 资产 / 报销）加小标题。

### 修复
- 概览走势图数据气泡里「支出」金额误带正号（如「支出 +23.19」），改为负号「-23.19」。

## [1.8.4] - 2026-07-07

### 新增
- **导入预览新增「导入账户 / 分类」映射区**（可折叠）：账单里要新建的账户 / 分类会集中列出，可给每一个**改名**，或**映射到已有的账户 / 分类**——映射后所有引用它的交易一次性全部改过去，不用逐条改。

### 优化
- 导入预览的交易列表改为**按日期分组**（从新到旧、带当日合计），与交易列表样式一致。
- 导入预览顶部汇总卡与下方交易列表左右**对齐**，不再宽窄不一。

## [1.8.3] - 2026-07-07

### 新增
- **首页概览卡可自定义**：进入收支统计页后点右上角「编辑」，可分别设置卡片上大数字、结余位与三个小卡片各展示什么数据（本月 / 今日 / 本周 / 本年 / 总的收入·支出·结余、日均消费·收入、总资产 / 负资产 / 净资产、待报销 / 已报销等），可设置走势曲线画支出 / 收入 / 结余，还能自定义卡片标题（留空显示「概览」），并一键恢复默认。

### 修复
- 首页概览走势窗口改为「累积展开」：当月 1 号起累积展示，到 8 号显示 1–14、15 号显示 1–21…… 直到覆盖整月，不再每 7 天跳成一段孤立的区间。

## [1.8.2] - 2026-07-07

### 新增
- **导入账单新增「导入预览」**：选好文件后不再直接入账，而是先进入预览页，用交易卡片列出即将导入的每一笔。可逐条排除 / 恢复、点进完整编辑（改金额、分类、账户、日期、备注等），确认后才真正导入；只会创建被保留交易实际用到的新账户 / 分类。

## [1.8.1] - 2026-07-07

### 修复
- 账单来源选择弹窗中图标与文字未垂直对齐的问题。

## [1.8.0] - 2026-07-07

### 新增
- **支持「一木记账」账单导入**（Excel .xls）。一木把账单与转账分开导出，对应两个独立入口——「一木记账 · 账单」和「一木记账 · 转账还款」，各选对应文件即可；选错文件会明确提示，不会误导入。
- 一木转账完整保留转出账户、转入账户与手续费。

### 优化
- 账单来源选择弹窗换上真实软件图标（支付宝 / 微信 / 薄荷记账 / 一木记账），更易辨认。

## [1.7.3] - 2026-07-07

### 优化
- 数据管理页重新分组、加小标题；「导出 / 导入」并入「本地备份」卡片，布局更清晰。

## [1.7.2] - 2026-07-07

### 优化
- 设置页按「通用 / 记账 / 关于」分组并加小标题，更易查找。

## [1.7.1] - 2026-07-06

### 优化
- 点「立即备份」时显示「备份中」转圈，避免点击后无反馈。

## [1.7.0] - 2026-07-06

### 新增
- 新增软件日志，数据保存失败时给出提示，方便排查。
- 记账提醒权限被拒时会提示，避免提醒静默失效。

### 隐私与安全
- 启用应用锁后开启防截屏：余额不再出现在系统「最近任务」缩略图或截图中。
- 配置 http 公网端点（WebDAV / AI）时，警告明文凭证外发风险。

### 备份
- 导入账单、调整余额也会触发自动备份与桌面小组件刷新。
- 自动备份更省资源；WebDAV 按保留份数自动清理旧备份。

### 修复
- 导入时拒绝把加密备份误当空备份，杜绝误清空数据。
- 修复周期记账按月 / 按年推进时的月末日期漂移（如每月 31 号）。
- AI 识别遇到异常字段时安全降级，不再报错中断。
- 英文界面下 AI 报错等不再夹带中文。
- 修复一批小问题：同账户转账、日期区间逆序、Excel 日期在夏令时地区偏移、图片重复压缩等。
