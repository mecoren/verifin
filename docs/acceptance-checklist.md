# Veri Fin 验收清单

## 当前已完成
- 首页支持快速记账、最近交易、收支统计入口、预算设置和日历日期交易入口。
- 交易支持本地保存、刷新保留、详情查看和删除确认。
- 资产页支持账户管理、分组管理、账户详情、账户报告和资产卡片背景自定义。
- 图片入口使用统一裁剪页：头像为 1:1，资产背景为宽幅封面比例。
- 看板页基于本地交易数据展示分类、排行、趋势和月度结构。
- 我的页支持个人资料编辑、账本管理、设置入口和本地数据状态展示。
- 设置页支持主题选择器、本地优先说明、GitHub CI 打包说明和初始化数据。
- 选择器统一使用底部弹出样式，破坏性操作使用确认弹窗。

## 验收重点
- 浅色和深色模式下，首页、资产、看板、我的四个 Tab 不应出现白屏、文本重叠或明显低对比内容。
- 首页卡片标题使用统一字号和字重，信息密度应接近“日历”卡片。
- 快速记账流程应完整闭环：FAB 输入金额、进入详情、保存后首页出现记录。
- 删除或隐藏账户后，快速记账详情页的账户选择中不应再出现该账户。
- 资产背景长按后应能选择线上图、自定义链接、本地图片和清除背景。
- 本地图片裁剪页应能调节缩放、水平位置和垂直位置，确认后显示裁剪结果。
- 头像编辑应复用同一裁剪页，保存后个人页展示新头像。
- 初始化数据前必须二次确认，确认后交易、账户、账本、预算、个人信息和主题偏好恢复默认。

## 暂未作为完成标准
- 云同步、登录账号、多端同步和远程备份。
- Android 本机打包；Android APK/AAB 由后续 GitHub CI 生成。
- 交易编辑、标签、项目、商家、退款和高级预算。
- Unsplash API 搜索图库；当前只使用预设线上图片和自定义图片链接。

## 后续优先级
- 继续精修整体视觉系统，重点是资产页、看板页和我的页的信息层级。
- 增加更真实的图标资源和分类图标体系。
- 建立 GitHub CI Android 打包流程。
- 增加导入、导出和本地备份能力。
